> Before making a change, find the affected component in column 1.
> Every component in its "Breaks" column must be updated in the same PR.

| If you change... | It breaks... | You must also update... |
|-----------------|-------------|------------------------|

| `app/middleware/tenant_middleware.py` | All data queries — potential data leak | `tests/test_tenant_isolation.py` MUST still pass |
| `app/database.py` (change session factory) | Every route using `Depends(get_db)` | All route files, all test fixtures in `tests/conftest.py` |
| `app/main.py` (add/remove middleware) | Security posture, rate limits, CORS | `tests/test_security_owasp.py`, `tests/test_contracts.py` |
