Preview Mobile App

Four ways to preview — choose based on what you have installed.

Download the ZIP, then run:

cd mobile
npm install
cp .env.example .env
npx expo start --web

Opens at localhost:8081. NativeWind v4 renders correctly on web — full styling, no emulator needed.

Local emulator (no account):

npx expo run:android   # needs Android Studio
npx expo run:ios       # needs Xcode (macOS only)

Shareable preview APK (free Expo account):

npm install -g eas-cli && eas login
eas build --platform android --profile preview

EAS gives you a QR-code download link to share with anyone. Build takes ~15 min. Full NativeWind v4 styling on real hardware.

Option 3 — Expo Snack

No NativeWind styling

Shows app structure and logic in a browser simulator. NativeWind className props are ignored in Snack — UI renders unstyled. Useful for checking navigation flow and data loading.

Open in Tab

Scan with Expo Go — note: NativeWind styling will not render. Use an EAS dev build for full styling.
